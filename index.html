Introducciòn

El curso diseño de sitios web se divide en tres Unidades, las cuales son:

1.Introduccion al diseño de sitios Web: En esta vemos que es y como funciona el sistema de control de versiones GIT y el uso de GITHUB para sicronizar nuestros proyectos a travez de internet y asi poder trabajar de forma colaborativa.

2. HTML5: Usado para estructurar y presentar el contenido para la web

3. CSS3:Ofrece la posibilidad de definir las reglas y estilos de representaciòn en diferentes dispositivos. Las hojas de estilo nos permite definir de manera eficiente la representaciòn de nuestras paginas web.

El OVI Diseño de sitios Web esta elanorado con el fin de dar a conocer informacion basica sobre el manejo de herramientas enfocadas al desarrollo Web como las son GIT, GITHUB, HTML5 Y CSS3. Esto se realiza a travez de lecturas, videos y actividades cortas y faciles de entender.

Objetivo General

El objetivo de este OVI es dar a conocer a los visitantes del sitio informacion basica sobre que es y como funciona GIT, GITHB, HTML5 y CSS3, y logrando dejar asi un apredizaje basico de cada uno.
